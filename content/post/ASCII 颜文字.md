## Preface {#preface}

在逛 Reddit 的时候总是可以看到一些简单的颜文字，比如: `:)`, `:D`.
觉得很有趣，写一篇文章总结一下。

| 字符符号      | 含义                |                 |         |
|-----------|-------------------|-----------------|---------|
| `:`           | 普通眼睛 `:)`       |                 |         |
| `;`           | 眨眼、暗示 `;D`     |                 |         |
| `D`           | 大笑 `:D`           |                 |         |
| `<`           | 担忧或撇开的眉毛 `<:-\vert` |                 |         |
| `B` `8`       | 眼镜 `8)`           |                 |         |
| `'`           | 眼泪 `:'(`          |                 |         |
| `#`           | 口罩 `:-#`          |                 |         |
| `&`           | 嘴巴被锁 `:-&`      |                 |         |
| `^`           | 尖鼻子 `;^)`        |                 |         |
| `P`, `p`, `b` | 吐舌头 `:b`         |                 |         |
| `<`           | 生气噘嘴 `D:<`      |                 |         |
| `%`           | 醉酒的眼睛 `%)`     |                 |         |
| `X`, `x`      | 紧闭双眼 `XD`       | 被封口  `:-X`   |         |
| `*`           | 嘟成一团，表示亲吻 `:*` | 眨眼或眼中有光 `*)` |         |
| `@`           | 极度愤怒、大喊大叫 `:@` | 玫瑰花朵 `@>-->--` |         |
| `-`           | 普通鼻子 `:-`       | 玫瑰花茎 `@>-->--` |         |
| `>`           | 下压的眉毛 `>:`     | 大笑或咆哮的变体 `X>` |         |
| `3`           | 猫嘴，卖萌或傲娇 `:3` | 动物耳朵或恶魔角 `3:-` |         |
| `{`, `}`      | 嘴唇上方的胡须 `:}` | 嘴唇发抖、委屈或不安 `:{` |         |
| `O`, `o`, `0` | 惊讶张大、惊呼 `:-O` | 天使光环 `O:-)` | 打哈欠  |
| \\(\vert\\)   | 嘴巴紧闭成直线      | 平行的冷漠眉毛  | 眼神空洞，无聊 |

主要参考 [List of emoticons](https://en.wikipedia.org/wiki/List_of_emoticons).


## Main {#main}


### 积极情感 {#积极情感}

微笑或快乐
: `:)` `:o)` `:]` `:3` `:c)` `:>` `=]` `8)` `=)` `:}` `:^)`

笑/咧嘴/戴着眼镜笑
: `:D` `8D` `xD` `X-D` `XD` `=D` `=-3` `=3` `B^D`

非常高兴或有双下巴
: `:-))`

喜极而泣
: `:'-)` `:')`

天使/圣人
: `O:)` `0:3` `0:)` `0;^)`


### 负面与悲伤情感 {#负面与悲伤情感}

皱着眉头/伤心
: `>:[` `:(` `:-c` `:c` `:-<` `:<` `:-[` `:[` `:{` `D:`

愤怒
: `:-||` `:@` `>:(`

哭泣
: `:'-(` `:'(`

恐怖/厌恶/悲伤/恐慌
: `D:<` `D:` `D8` `D;` `D=` `DX` `v.v` `D-':`

邪恶
: `>:)` `}:)` `3:)`

生病
: `:#` `:&` `:X`


### 面部小动作与复杂情绪 {#面部小动作与复杂情绪}

惊讶/震撼/打哈欠
: `:O` `o_O` `o_0` `o.O` `8-0`

亲吻
: `:*` `:^*`

眨眼/傻笑
: `;)` `*)` `;]` `;D` `;^)` `:-,`

伸出舌头/俏皮/厚脸皮
: `:P` `xp` `XP` `:p` `=p` `:b`

怀疑/恼火/犹豫不决/忐忑不安
: `>:\` `>:/` `:-/` `:-.` `:/` `:\` `=/` `=\` `:L` `=L` `:S`

板着脸/毫无表情/优柔寡断
: `:|`

尴尬/红脸
: `:$`

被封口/戴牙套
: `:X` `:#`

舌头打结
: `:-&` `:&`

喝醉
: `%)`

派对装
: `#-)`

无聊/打哈欠
: `|-O`
