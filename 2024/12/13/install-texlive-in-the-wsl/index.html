Install-texlive-in-the-wsl

我在这一篇文章会介绍一下三种在 windows 编辑 latex 的配置。可以根据自己的需要选择。

  • overleaf: latex 可以在这个网站实现在线编辑，而无需任何配置，但是由于是部署在国外的网站，在国内网络连接不够稳定，所以可能需要魔法。另一方面速度也会比较缓慢，而且没有那么灵活。如果官网里面的模板足够满足要求，这是一个非常好的选择。
  • windows 本地使用 vscode 编辑。vscode 是一个微软官方开发的文本编辑器，里面的插件非常丰富，几乎可以满足一切需求。不过这个方案可能有以下问题：
    • windows 本地部署虽然有图形化界面，但是受限于 windows 环境变量的问题，以及复杂有些臃肿的系统，可能出现环境变量丢失或者其他难以排查的问题。
  • wsl + vscode：wsl(windows subsystem of Linux) 同样也是微软官方开发的工具（属于一种虚拟机，用来在 windows 的环境下模拟 Linux 系统，这几乎是 windows 下模拟 Linux 的最佳选择）。虽然是虚拟机可能性能不够好，但是 Latex 在 Linux 的环境下运行反而快很多。这个方案可能有以下问题：
    • 需要一点计算机的常识。
    • 安装 vscode-serve 的时候可能需要魔法（我第一次配置没有出现问题，之后重装系统结果配置的时候就出现了莫名的问题，无法解决，最后使用魔法就 OK 了）
    • 默认安装在 C 盘，请先确认你的 C 盘剩余储存充足（最小化安装至少 4G）。
