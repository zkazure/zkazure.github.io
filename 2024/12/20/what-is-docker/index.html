What Is Docker

Kazure Zheng
2024-12-20
software
docker

Container

Before we learning the docker, we have to know what is docker. In the development of applications, we need to transmit our appliciton between different collegue and different computer. However, in the different computer the environment is different, the application may go well on your computer, and go wrong on your collegues’. In order to solve the problem, someone choose to copy the whole operating system, but it will consume much more resource. So container arised at the historic moment, it packs up the application and its bin or lib together in the container, so that you are able to run application on different computer.

Docker

So docker is accutually a project writed with GO programming language to achieve container technique. It just like a function in your mobile phone to create mirror of application in your phone.
