list in the python

  • list(): break into list temporaliy.

  • index(): return the first position of a figure in a breakable varibles.

  • count(): count the appear times of the figures.

  • append(): add a terms at the end

  • extend(): add the terms in the new list to the list

  • pop(): remove the specific position terms, and return the removed term.

  • remove(): remove the specific term in the list

  • reverse(): as the name say

  • sort(list, key = len, reverse= True): change the sort of list

  • sorted(): the same as sort, but it doesn’t change the sort of the list and return sorted list
