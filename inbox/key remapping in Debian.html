Preface

使用 Emacs 这样的工具，虽然极大提升了效率，但是大量的修饰键，容易 rsi。因此我尝试了一些社区主流的 key remapping。

Main

有两种方法，

Global

修改 /etc/default/keyboard, 

XKBOPTIONS="ctrl:swapcaps, ctrl:swap_ralt_rctl"

交换了左 ctrl 和 CapsLock，右 ctrl 和右 alt。

Temporal

setxkbmap -option "ctrl:swapcaps"
