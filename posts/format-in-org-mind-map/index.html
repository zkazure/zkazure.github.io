Raster Link to heading

GIF(Graphics Interchange Format) Link to heading

  • low quality
  • support animation and transparent

PNG(Portable Network Graphics) Link to heading

  • high quality
  • support transparent
  • suitable for web
  • not ideal for print

JPEG(Joint Photographic Experts Group) Link to heading

  • lossy compression
  • multiple colors, ideal for photographs
  • good for web

TIFF(Tagged Image File Format) Link to heading

  • best quality
  • support layers, transparency.
  • for professional photography or print
  • too large

Vector Link to heading

EPS(Encapsulated PostScript) Link to heading

  • common used in printing and design
  • scalable without losing quality
  • being replaced

SVG(Scalable Vector Graphics) Link to heading

  • the same as EPS
  • editable with code
  • support animation and interactivity

don’t need introductions
