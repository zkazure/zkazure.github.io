Difference of the format in the org-mind-map

March 6, 2025 · zkazure

Raster#

GIF(Graphics Interchange Format)#

  • low quality
  • support animation and transparent

PNG(Portable Network Graphics)#

  • high quality
  • support transparent
  • suitable for web
  • not ideal for print

JPEG(Joint Photographic Experts Group)#

  • lossy compression
  • multiple colors, ideal for photographs
  • good for web

TIFF(Tagged Image File Format)#

  • best quality
  • support layers, transparency.
  • for professional photography or print
  • too large

Vector#

EPS(Encapsulated PostScript)#

  • common used in printing and design
  • scalable without losing quality
  • being replaced

SVG(Scalable Vector Graphics)#

  • the same as EPS
  • editable with code
  • support animation and interactivity

PDF(Portable Document Format)#

don’t need introductions
