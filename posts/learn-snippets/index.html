snippets

In the doom emacs the defaut directory is ~/.doom.d/snippets/ . And in the the file should be divided into different derctory to triger different file in the corresponding mode.

front-matter

# -*- mode: snippet -*-
# name: the name to search
# key: the key word to be completed by <tab>
# condition: if here is nil, the snippet will not be trigger
# --

variables

main

${1:default_value}
${2:label}
${2:same}
;; 插入 snippets 后可以使用 <tab> 按顺序跳转。
;; 里面的默认值可以用作提示
;; 还可以插入相同的值
${0}
;; 这个是最后的光标位置
${cursor}
;; 使用后会自动跳转，也就意味着不能够修改其他的${数字}

advance

${CILPBOARD}
;; 从当前的剪切版中获取信息
${SECLRCTION}
;; 从当前选择的内容中获取信息
;; 但是似乎难以决定输入的位置
;; 不如使用 ${CILPBOARD}
