Doom emacs

emacs

Basic

  • windows manager

    • SPC w v: windows vertical split
    • SPC w s: windows split
    • SPC w hjkl: windows jumped to left/down/up/right.
    • SPC w HJKL: windows move.
    • SPC w b: windows jumped to bottom right.
    • SPC w t: windows jumped to top left.
    • SPC w w: windows jumped to next.
    • SPC w W: windows jumped to previous.
    • SPC w c: windows closed.
    • SPC w x: windows exchange.
    • SPC w +-<>: windwos change size.
    • SPC w =: windows balance.
    • SPC w m m: windows maximize the current windows.
    • SPC w u: undo the windows.

    Practice.

    • create four windows in four directions.
    • jumped between windows
    • move and exchange the windows.
    • what the order in the windows previous and next.
    • jumped between top left to right bottom.
    • change the size of the windows and then balance them.
    • maximize the current windows, then undo it.
    • close all windows.

  • edit file

    • SPC f r: find rescent file.
    • SPC f f: find file. (also SPC . )
    • SPC f F: find From here.
    • SPC f D: file Delete.
    • SPC f s: file save. (to replace :w)
    • SPC f S: file Save as …
    • SPC f R: file Rename/move.
    • SPC f C: file copy

    Now let’t do some practice.

    • create a file in the ~/test/file.org
    • input anything then save it.
    • copy the file as file-cp.org in two ways.
    • find the copy file in current diretory.
    • rename one file and move another file to ~/test/move/
    • find recent origin file.
    • delete all the file you create.

Advanced

  • bookmark

    • SPC RET: open the bookmark

    if you didn’t have one, input the name, RET will create a bookmark where you cursor in.

基本结构

    preperty

    edge

使用 C-c C-c 自动运行导出显示图片

Graph type

  1. digraph：有向
  2. graph: 无向

node

  1. digraph: a -> b;
  2. graph: a -- b;
  3. 如果包含特殊符号，使用双引号。 "x.y"

edge

  1. 连接多个节点 a -> {b; c; d};
  2. 反向 a -> c [dir=back];

properties

    label = "picture name"
    graph[bgcolor=lightblue];
    rankdir=LR
    node[shape=box];
    edge [color=green]
    a [color=blue; style=filled; fontcolor=green];
    a -> b [color=red; label=A; dir=back];

snippets

In the doom emacs the defaut directory is ~/.doom.d/snippets/ . And in the the file should be divided into different derctory to triger different file in the corresponding mode.

front-matter

# -*- mode: snippet -*-
# name: the name to search
# key: the key word to be completed by <tab>
# condition: if here is nil, the snippet will not be trigger
# --

variables

main

${1:default_value}
${2:label}
${2:same}
;; 插入 snippets 后可以使用 <tab> 按顺序跳转。
;; 里面的默认值可以用作提示
;; 还可以插入相同的值
${0}
;; 这个是最后的光标位置
${cursor}
;; 使用后会自动跳转，也就意味着不能够修改其他的${数字}

advance

${CILPBOARD}
;; 从当前的剪切版中获取信息
${SECLRCTION}
;; 从当前选择的内容中获取信息
;; 但是似乎难以决定输入的位置
;; 不如使用 ${CILPBOARD}
