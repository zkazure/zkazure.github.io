  1. 数组的下标都是从零开始
  2. 内存空间地址连续
#include <iostream>
using namespace std;
void test_arr(){
    int array[2][3] = {
    {0, 1, 2},
    {3, 4, 5}
    };

    cout << &array[0][0] << " " << &array[0][1] << " " << &array[0][2] << endl;
    cout << &array[1][0] << " " << &array[1][1] << " " << &array[1][2] << endl;
}

int main(){
    test_arr();
    return 0;
}

输出的地址

0x7ffea7a6d920 0x7ffea7a6d924 0x7ffea7a6d928
0x7ffea7a6d92c 0x7ffea7a6d930 0x7ffea7a6d934

首先储存的是 int 类型的数据，所以是4个字节，显然连续。
