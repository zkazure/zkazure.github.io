how to add a table of content to hugo

March 11, 2025 One-minute read
Work • Hugo
  1. Enable official TOC
  2. Set up in the html
    1. example in hugo-coder

Here I use the official way.

Enable official TOC Link to heading

In the hugo.toml, add blow:

[markup]
    [markup.tableOfContents]
        endLevel = 4
        ordered = true
        startLevel = 2

Set up in the html Link to heading

In the theme/<your-theme>/layouts/posts/single.html, in the content, add blow:

<div class="toc-container" >
{{ .TableOfContents }}
</div>

example in hugo-coder Link to heading

<div class="post-content">
{{ if .Params.featuredImage }}
        <img src="{{ .Params.featuredImage | relURL }}" alt="Featured image"/>
{{ end }}

        <!-- TOC -->
        <div class="toc-container" >
        {{ .TableOfContents }}
        </div>
        <!-- TOC -->

{{ .Content }}
</div>
