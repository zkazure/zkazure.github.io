## Backup and Recover Wsl

  1. Backup

```zsh wsl –shutdown wsl –export Debian D:\WSL\debian.tar wsl –unregister Debian

```

  1. recover

```zsh wsl –import Debian D:\WSL\Debian D:\WSL\debain.tar ```

  1. in the wsl

```zsh su user sudo su cp /etc/wsl.conf /etc/wsl.config.bk nvim /etc/wsl.conf ```

  1. add

```zsh [user] default=user ```

  1. reboot

### Docker

#### Preparation

```bash sudo apt update sudo apt install -y apt-transport-https ca-certificates curl gnupg lsb-release curl -fsSL https://mirrors.aliyun.com/docker-ce/linux/ubuntu/gpg | sudo gpg –dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg

```

#### Software Configuration

```bash $ echo \ “deb [arch=amd64 signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://mirrors.aliyun.com/docker-ce/linux/ubuntu \ $(lsb_release -cs) stable” | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

```

#### Finally

```bash sudo apt update sudo apt install docker-ce docker-ce-cli containerd.io ```
