Headline#

z z: scoll to center SPC m h: create a healine under this current healine SPC m i: create item C-RET: insert the same item. TAB: to flod and unfold. S-TAB: unfold. z c: close z o: open z M: close all z R: open all g j/k: go to the next same level g h/l: go to the next or last level. C-RET: the new level the same as the current level M-h/l: change the level

M-j/k: move the whole title and the content, M-S-j/k: move the current line.

v a e: visual seclet a element d a e: delete an element v a R: visual seclect a subtree. d a R: delete a subtree y i R: yank a subtree

SPC-i-s: insert a snippets. SPC-m-t: the state of todo SPC m d s: schedule.

  • check C-c C-c change

SPC m b -: make headline S-RET: Duplicate a file in the cell below
