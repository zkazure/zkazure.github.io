# install kaiwu sdk


1. install python3.8
2. change the path
3. check version
```
python --version
```
4. install sdk
```
python -m pip install --upgrade pip
pip3 install kaiwu-1.0.3-py3-none-any.whl -i https://pypi.tuna.tsinghua.edu.cn/simple

```
