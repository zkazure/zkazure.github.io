## in the Debian
1. install
```
sudo apt update
sudo apt install neofetch
```
2. config
```
vim ~/.zshrc
```
3. add neofetch at the end.

## in the windows

1. install scoop
```
iwr -useb get.scoop.sh | iex
scoop install git
```
2. set the scoop bin path(usually C:\Users\YourUserName\scoop\apps\scoop\current\bin) to the environment path

3. install neofetch
```
scoop install neofetch
```
4. config
```
vim ~/.zshrc
```
5. add "neofetch" at the end
