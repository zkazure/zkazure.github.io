finding specific function from the perspective of linear algebra

y

Example

Let $ F(x) $ be defined for all real numbers
except at $ x=0 $ and $ x=1 $,
and satisfy the equation
$ F(x) + F( \frac{x-1}{x}) = 1+x $.
Find the function $ F(x) $

Solution

\begin{equation}
  \label{eq:1}
  F(x) + F(\frac{x-1}{x}) = 1+x
\end{equation}

make $ x = \frac{x-1}{x} $, have

\begin{equation}
  \label{eq:2}
  F(\frac{x-1}{x}) +F(- \frac{1}{x-1}) = \frac{2x-1}{x}
\end{equation}

make $ x = - \frac{1}{x-1} $ in \cite{eq:2}, have

\begin{equation}
  \label{eq:3}
  F(- \frac{1}{x-1}) + F(x) = \frac{x-2}{x-1}
\end{equation}

According to \cite{eq:1}, \cite{eq:2}, \cite{eq:3}, have

\begin{equation}
  \label{eq:4}
  F(x) = \frac{x ^3-x ^2-1}{2x(x-1)}
\end{equation}

Linear algebra perspective

Found that you finally attained 3 equation with functions with 3 different variables.

Why not consider it as the matrix in linear algebra where you have to have 3 different linear equations to obtain actual values of 3 variables.

Now we can construct the correct matrix to solve every problems of finding specific function.

How many transforms we need to find the specific function

  • simplest condition

    If we just need two times.

    if we have $ F(a) + F(b) = c_1 $
make $ a=b $, then $ F(b) + F(a) = c_2 $

\begin{bmatrix}
  1 & 1 & $ c_1 $ \\
  1 & 1 & $ c_2 $
\end{bmatrix}

    It is easy to know the matrix is inconsistent.

  • different coefficient

    \begin{bmatrix}
  $ t $ & 1 & $ c_1 $ \\
  1 & $ t $ & $ c_2 $
\end{bmatrix}

    It is consistent.

  • how long the transformation

    • return to example

      \[ x \to \frac{x-1}{x} \to - \frac{1}{x-1} \]

\[ \frac{x-1}{x} \to - \frac{1}{x-1} \to x \]

      It is easy to notice that it construct a loop.

    • extended idea

      Let consider $ x_i (i \in R) $ as one variable in the function. And $ x_i \ne x_j (i,j \in R, i \ne j) $

      If you have only 2 equation, then you should write the tranformation down.

      \[ x_1 \to x_a \to \dots \to x_2 \]

\[ x_2 \to x_b \to \dots \to x_1 \]

      You will find the specific solution.

    • question

      What if there are other loops?

      Yes, it is possible and tricky. However, there is only one loop, if we are going to find the specific function. Because there is only one solution.
